❓ FAQ

Frequently Asked Questions

Quick answers to common questions about this frontend package.

What is included in the delivery? +
You receive the complete HTML, CSS, and JavaScript source files — organized, commented, and ready to use. No frameworks, no dependencies, just clean vanilla code.
Is backend or hosting included? +
No, this is a frontend-only package. Backend, database, hosting, and domain setup are not included. The website runs entirely in the browser.
Can I customize the design? +
Absolutely! You can change colors, fonts, images, and content easily. The CSS uses variables, making theme customization quick and simple.
How long does delivery take? +
Basic: 2 days. Standard: 3 days. Premium: 5 days. Timelines may vary slightly based on complexity and revisions.
